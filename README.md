To-Do-List web-application developed for task management.

Features
---
- User login.(Sign In)
- Create To-do-List.
- Update(change title), delete To-do-Lists.
- Create To-do-List Items into To-do-Lists.
- Update, Delete and Complete actions for Todo List Item.
- Filter(by status).
- Search to name into Todo List.
- Drag 'n' Drop


Installation
---
Clone down this repository. You will need node and yarn installed globally on your machine.

Installation:

yarn install

To Run Test Suite:

To Start Server:

yarn start

To Visit App:

localhost:3000

Application Architecture
---
- IDE: WebStorm
- Programming Language: TypeScript
- Using React, Redux
- Formik
- Material Ui
- FontAwesome
- Axios
